# Calculadora Web 🧮

Aplicación web desarrollada con HTML, CSS y JavaScript que permite realizar operaciones matemáticas básicas.

## 🚀 Funcionalidades
- Suma
- Resta
- Multiplicación
- División
- Interfaz interactiva

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## ▶️ Cómo usar
1. Abrir el archivo `index.html` en el navegador
2. Usar los botones para realizar operaciones
